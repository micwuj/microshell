# microshell
This code is a simple implementation of a command-line interface shell. The program is written in C language and is designed to accept Linux commands, as well as the custom commands defined in this program.

The program uses several C libraries such as stdio.h, unistd.h, sys/wait.h, string.h, and errno.h, among others. These libraries provide functionality such as input and output, process control, string manipulation, and error handling.
